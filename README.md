# sample_web_speech_api_in_electron

## Install

```
git clone git@github.com:itkrt2y/sample_web_speech_api_in_electron.git
cd sample_web_speech_api_in_electron
npm install
```

## Build

### build all assets

```
npm run build
```

### watch files

```
npm run watch
```

## Run Application

```
npm start
```
